# 📊 Social Media Analysis: Tweet "fufufafa"

Analisis terhadap sentimen terhadap kata "fufufafa" di twitter menggunakan pendekatan data mining dan Natural Language Processing (NLP).

## 🎯 Tujuan Proyek
- Melakukan analisis sentimen (positif, negatif, netral) dari tweet
- Mengidentifikasi topik atau kata kunci yang paling sering muncul

## 🛠 Tools & Teknologi

- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **NLP Library** (NLTK / TextBlob / Sastrawi untuk bahasa Indonesia)
- **Visualisasi** (WordCloud, Bar Chart, Time Series Plot)
- **Environment** (Jupyter Notebook / VS Code)
